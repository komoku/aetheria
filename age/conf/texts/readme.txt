AETHERIA GAME ENGINE: PLAYER'S KIT


ESPAÑOL (SEE BELOW FOR ENGLISH)

Aetheria Game Engine permite jugar a todo tipo de juegos basados en texto, mono y multijugador; en sistemas Windows, Linux y Mac.

Para ello, tienes que tener instalado Java versión 1.6 o superior. Si no lo tienes, puedes obtenerlo gratis en www.java.com 

Ejecuta el fichero simpleage.bat en Windows (simpleage.sh en Linux, simpleage.command en Mac) y escoge el mundo que quieres jugar.

ENGLISH

Aetheria Game Engine allows you to play all kinds of text-based games, single and multiplayer; on Windows, Linux and Mac systems.

To do so, you must have Java (version 1.6 or later) installed. If you do not have it, you can obtain it for free at www.java.com

Execute the file simpleage.bat under Windows (simpleage.sh on Linux, simpleage.command on Mac) and choose the world you would like to play.
